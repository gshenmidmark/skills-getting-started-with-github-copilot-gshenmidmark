# Skills: Getting Started with GitHub Copilot

Exercise: Get started using GitHub Copilot - Chapter 03

## Chapter 03: Building a School Activities Management System

This repository demonstrates building a FastAPI application with GitHub Copilot for managing extracurricular activities at Mergington High School.

### Features Built with Copilot

- 🏫 **Activity Management**: View and manage school extracurricular activities
- 👥 **Student Signup**: Students can sign up for activities with email validation
- 📊 **Real-time Tracking**: Live participant counts and availability status
- 🎨 **Modern Interface**: Responsive web interface with interactive features
- 🔧 **REST API**: Full API with automatic documentation

### Quick Start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python src/app.py
   ```

3. Visit http://localhost:8000 to use the web interface

### API Documentation

- Interactive API docs: http://localhost:8000/docs
- Alternative docs: http://localhost:8000/redoc

See [src/README.md](src/README.md) for detailed API documentation.
